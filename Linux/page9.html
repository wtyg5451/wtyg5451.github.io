1. 将node-v18.15.0-linux-x64.tar.xz文件和three-gltf-viewer文件夹，通过远程连接工具, 放入Downloads文件夹
2. 复制到root根目录
3. 修改node-v18.15.0-linux-x64.tar.xz文件权限
4. 两次解压并更名
xz -d node-v18.15.0-linux-x64.tar.xz
tar xvf node-v18.15.0-linux-x64.tar -C /opt/
mv node-v18.15.0-linux-x64 nodejs
5.设置软链接
ln -s nodejs/bin/npm /usr/local/bin
ln -s nodejs/bin/node /usr/local/bin
ln -s nodejs/bin/npx /usr/local/bin
6.查看版本号
node --version
npm --version
7. 进入three-gltf-viewer文件夹，调试运行项目
npm install
npm run dev
8.在linux自带的浏览器中查看
http://localhost:3000
