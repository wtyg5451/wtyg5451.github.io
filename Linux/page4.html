1. 从桌面模式切换到终端模式
按CTRL+ALT+T
2. 切换到超级用户执行权限
sudo -i
如果有密码，输入密码。执行该命令后提示符变为“#”而不是“$”。
3. 查看当前路径命令
pwd
4. 显示当前文件夹的内容
ls
5. 查看ip地址
ip a
6. 更改网络配置
ip地址: 172.16.18.2**(其中**为自己的两位学号）
netmask子网掩码: 255.255.255.0
gate way网关: 172.16.18.254
DNS（域名服务）： 220.189.127.107
重置网卡后，更改网络连接为：由原来的NAT模式为桥接模式，此时出现hotspot login登录，输入校园网登录帐号和密码，linux系统internet已连通
7. 测试internet网络是否连接
ping baidu.com
