WeBASE安装
#mysql中不用自己创建表，fiscobcos也不用提前装
#一键脚本，会自己安装fisco bcos
#不要使用root权限去安装，所有需要用到root权限的地方都加了sudo

1、系统环境搭建
采用ubuntu20.04
sudo apt-get update
sudo apt install -y openssl curl
sudo apt-get install software-properties-common

2、JAVA环境配置
sudo apt install -y default-jdk
#添加java_home
sudo gedit /etc/profile
#添加以下变量
export JAVA_HOME=/usr/lib/jvm/default-java
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
#保存后命令行执行，引入最新的变量
source /etc/profile

3、mysql
sudo apt install mysql-server-8.0
sudo apt install mysql-client-8.0

4、查找mysql用户名和密码，后面的配置文件中用的到
sudo cat /etc/mysql/debian.cnf

5、python3环境
sudo apt-get install -y python3-pip
sudo pip3 install PyMySQL

6、执行一键安装webase
wget https://osp-1257653870.cos.ap-guangzhou.myqcloud.com/WeBASE/releases/download/v1.5.3/webase-deploy.zip
unzip webase-deploy.zip
cd webase-deploy

7、修改webase基础配置
gedit common.properties
