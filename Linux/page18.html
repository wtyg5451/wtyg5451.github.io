1. 将node-v20.11.1-linux-x64.tar.xz文件，通过远程连接工具, 放入Downloads文件夹
2. 复制到root根目录
3. 修改node-v20.11.1-linux-x64.tar.xz文件权限
4. 两次解压并更名
tar xvf node-v20.11.1-linux-x64.tar.xz -C /opt/
mv node-v20.11.1-linux-x64.tar.xz nodejs
5.设置软链接
ln -s nodejs/bin/npm /usr/local/bin
ln -s nodejs/bin/node /usr/local/bin
ln -s nodejs/bin/npx /usr/local/bin
6.查看版本号
node --version
npm --version
7. 安装truffle
npm install truffle
