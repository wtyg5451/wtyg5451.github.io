1. 将下载的 jdk-8u381-linux-x64.tar.gz通过远程连接工具, 放入/opt/目录, 然后解压
tar -zxvf jdk-8u381-linux-x64.tar.gz -C /opt/
2. 在opt文件夹中，将文件夹jdk1.8.0_2501改名为jdk.
mv jdk-8u381-linux-x64 jdk
3. 配置环境变量
gedit /etc/profile
添加以下几行
export JAVA_HOME=/opt/jdk
export PATH=$JAVA_HOME/bin:$PATH
export CLASSHOME=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
使环境变量生效
source /etc/profile
5、检查java版本
java -version
