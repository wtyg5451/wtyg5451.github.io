1. 用winscp上传 fisco-bcos文件和buildchain.sh文件到指定的fisco文件夹
2. 在fisco目录下执行以下指令，生成一条单群组4节点的FISCO链
bash build_chain.sh -l 127.0.0.1:4 -p 30300,20200,8545 -e ./fisco-bcos -v 2.8.0..
3.启动所有节点
bash nodes/127.0.0.1/start_all.sh
3. 检查进程是否启动
ps -ef | grep -v grep | grep fisco-bcos
4. 查看节点node0至node3链接的节点之间是否连接
tail -f nodes/127.0.0.1/node0/log/log* | grep connected
tail -f nodes/127.0.0.1/node1/log/log* | grep connected
tail -f nodes/127.0.0.1/node2/log/log* | grep connected
tail -f nodes/127.0.0.1/node3/log/log* | grep connected
